# 📊 Customer Churn Analysis and Prediction

## 🚀 Project Overview

This project focuses on predicting customer churn using machine learning techniques. Customer churn refers to when a customer stops using a company's service. Identifying churn patterns helps businesses take proactive steps to retain customers.

The project involves an end-to-end data science pipeline, including data preprocessing, exploratory data analysis (EDA), model building, evaluation, and feature importance analysis.

---

## 📁 Dataset

The dataset used in this project is the **Telco Customer Churn Dataset**, which contains customer demographics, account information, and service usage details.

### Features include:

- Customer demographics (gender, senior citizen)
- Account information (tenure, contract type)
- Services (internet service, phone service)
- Billing information (monthly charges, total charges)
- Target variable: **Churn (Yes/No)**

## 🔍 Exploratory Data Analysis (EDA)

- Checked for missing values and data types
- Analyzed customer distribution and churn patterns
- Visualized relationships between features and churn
- Identified important trends affecting customer retention

## 🧹 Data Preprocessing

- Handled missing values
- Converted data types
- Encoded categorical variables using one-hot encoding
- Feature scaling 

---

## 🤖 Machine Learning Models

The following models were implemented:

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

---

## ⚙️ Model Evaluation

Models were evaluated using:

- Accuracy  
- Precision  
- Recall  
- F1-score  
- Confusion Matrix  

---


## 📈 Results

- The model achieved good accuracy in predicting customer churn  
- Key factors affecting churn include:  

  - Contract type  
  - Monthly charges  
  - Tenure
